## Project Overview

DroneMedic is an AI-powered system for delivering medical supplies to remote and disaster-hit areas. It uses an LLM coordinator and simulated flight control to handle multi-stop routing while adapting to weather, no-fly zones, and urgent needs — optimising last-mile healthcare delivery with minimal human input.
